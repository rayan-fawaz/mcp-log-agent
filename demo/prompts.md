# Example Prompts for LogMCP

Use these prompts in VS Code Agent Mode to interact with your logs through AI.

## Log Analysis & Debugging

```
Get logs from the NA region between 1768473996370 and 1768473996400
```

```
Show me all errors from the EU region in the last hour and summarize the patterns
```

```
What's causing the 401 authentication failures in the NA region?
```

```
Find all requests that took longer than 1 second and identify the slow endpoints
```

```
Analyze the 500 errors in AP region and suggest possible root causes
```

## Customer Issue Investigation

```
Find all logs for orgID=org-12345 and summarize any issues
```

```
What errors is customer org-eu-002 experiencing today?
```

```
Correlate the payment failures with any upstream service issues
```

## Proactive Monitoring

```
Are there any unusual patterns or anomalies in the EU region logs?
```

```
Check if the error rate has increased compared to the previous time window
```

```
Summarize the health status across all regions
```
